★ Recommended New Features

Priority 1

Proactive Retention Agent — churn & cancellation prevention

Our AI today is mostly reactive (waits to be asked). The biggest untapped value is going proactive.

What it does

  • Nightly job scores at-risk / lapsing customers → drafts a personalized win-back message with a targeted RCN reward bonus as the incentive.
  • Detects cancellation-risk bookings and suggests a retention offer before they fall through.
  • Surfaces everything confirm-gated in the existing marketing assistant — nothing sends without owner approval.

Why it wins

  • Directly drives revenue and uniquely uses our token economics as the retention lever.
  • Mostly wiring together existing pieces (anomaly detection, lapsed-customer insights, follow-up handlers, campaign tools) — not new infrastructure.
Builds on: Insights tools, Anomaly detector, Marketing assistant, RCN rewards
Effort: Medium
ROI: Highest
Priority 2

Semantic Search via Embeddings — recommendation-quality upgrade

Customer repair chat and the marketplace currently use keyword/tag matching — the weakest link in recommendation quality.

What it does

  • Add pgvector to our existing PostgreSQL; embed services, FAQs, and past conversations.
  • Back the diagnostic chat's service-search tool and the marketplace with vector (meaning-based) search.

Why it wins

  • One upgrade lifts many features at once — better diagnoses, search, and FAQ matching.
  • No new database required; low risk, foundational investment.
Builds on: Existing Postgres, customer chat search tool, marketplace
Effort: Medium
ROI: High (foundational)
Priority 3

Voice Booking Agent — inbound phone calls

Repair shops live on phone calls. We already have the voice building blocks.

What it does

  • A real-time voice agent answers calls, diagnoses the issue, quotes a service, and books an appointment — handing off to a human on escalation.

Why it wins

  • Genuinely novel for this vertical; turns missed calls into booked revenue.
  • Reuses our Whisper, TTS, dispatch classifier, and booking tools.
Builds on: Whisper, TTS, voice dispatch, booking/availability tools
Effort: Larger (telephony)
ROI: High — flagship differentiator

→ Recommended Path

Start with the Proactive Retention Agent — best ROI, reuses our infrastructure, and showcases the rewards angle that makes RepairCoin different. Semantic Search is the best foundational investment if we'd rather strengthen the base first — and the two pair well, since better search makes the retention agent's recommendations sharper.
Suggested sequence: 1) Retention Agent (quick revenue win) → 2) Semantic Search (foundation) → 3) Voice Booking Agent (flagship).
