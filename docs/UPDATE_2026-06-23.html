1 Fraud & Abuse Detection — full feature Shipped

Built the platform's first Admin AI feature start-to-finish — protecting the token economy from reward abuse, collusion, and review manipulation.

Detection engine (7 rules)

  • A nightly scan flags suspicious patterns across the transaction ledger + reviews: concentrated issuance, rapid earn→redeem cycling, issuance spikes, review brigading, shop↔wallet self-dealing, cross-shop redemption anomalies, and rating manipulation.
  • Each finding is scored 0–100 by severity, deduplicated, and stored for review.

Admin "Trust & Safety" review queue

  • New admin tab: severity-sorted findings with summary counts, status filters, the metrics behind each finding, and a plain-English explanation.
  • Actions per finding: Investigate, Dismiss, or Confirm & Suspend.

Enforcement & intelligence

  • Confirm & Suspend reuses the existing audited shop/customer suspension (targeted, not a platform-wide freeze).
  • AI-written explanations — high-severity findings get a concise Claude-generated summary for the reviewer (with a templated fallback).
  • High-severity alerts — each serious new finding raises an admin alert so nothing is missed.

2 Platform Health Copilot Shipped (Phase 1)

A second Admin AI feature: an "ask the platform" assistant — the platform-wide version of the insights chat shop owners already have.

  • Admins can ask in plain English — "give me a platform overview", "RCN issued vs redeemed this month", "how healthy are shop subscriptions?" — and get tool-backed answers with data cards.
  • Launches with 3 platform tools: platform overview (shops / customers / RCN totals), token economy (issued vs redeemed + RCG tier mix), and subscription health (shops by status + active subscriptions).
  • New "Platform Copilot" tab in the admin dashboard; reuses the proven AI chat engine and data-card rendering.
