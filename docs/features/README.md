## ü™ô Dual-Token System

### RCN (Utility Token)
- **Purpose**: Rewards and redemptions
- **Value**: 1 RCN = $0.10 USD
- **Usage**: 
  - Customers earn from repairs
  - Customers redeem at shops
  - Shops purchase from admin

### RCG (Governance Token)
- **Purpose**: Shop tiers and governance
- **Supply**: 100M fixed
- **Usage**:
  - Shop tier qualification
  - DAO voting rights
  - Revenue sharing (10% to stakers)

## üë• User Tiers

### Customer Tiers
- **Bronze**: 0 RCN bonus
- **Silver**: +2 RCN bonus per transaction
- **Gold**: +5 RCN bonus per transaction

### Shop Tiers
- **Standard**: 10K+ RCG
- **Premium**: 50K+ RCG (better RCN pricing)
- **Elite**: 200K+ RCG (best RCN pricing)
