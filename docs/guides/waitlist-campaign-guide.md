# Waitlist Campaign Pages - Marketing Guide

## Overview

RepairCoin has campaign-specific waitlist pages that automatically track where signups come from. Just share the right link — tracking is automatic.

## Campaign Links

| Campaign | Link | Use For |
|----------|------|---------|
| Default | `repaircoin.ai/waitlist` | Website links, general use |
| Organic | `repaircoin.ai/waitlist/organic` | Social posts, blogs, SEO content |
| Facebook | `repaircoin.ai/waitlist/fb` | Facebook & Instagram ads |

That's it. No extra parameters needed. The system automatically tracks which page each visitor lands on.

## How to Use

1. **Pick the right link** based on your campaign type
2. **Use it as your ad destination** or share it in your post/email
3. **Check results** in Admin Dashboard → Waitlist tab

### Examples

- **Facebook ad** → use `https://repaircoin.ai/waitlist/fb`
- **Instagram ad** → use `https://repaircoin.ai/waitlist/fb`
- **Twitter post** → use `https://repaircoin.ai/waitlist/organic`
- **Blog article** → use `https://repaircoin.ai/waitlist/organic`
- **Email newsletter** → use `https://repaircoin.ai/waitlist/organic`
- **Website button** → use `https://repaircoin.ai/waitlist`

## Where to See Results

Go to **Admin Dashboard → Waitlist tab**. You'll see:

- **Campaign Performance section** — visits, signups, and conversion rate per source (Direct / Organic / Facebook)
- **Source filter** — filter the entries table by source
- **Source badge** — each entry shows where it came from

## Need a New Campaign Page?

If you need a new landing page (e.g., `/waitlist/tiktok` or `/waitlist/google`), ask the dev team. It only takes adding one config entry — no design work needed.
