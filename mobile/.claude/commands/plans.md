Create a detailed implementation plan for the following task.

## Task

$ARGUMENTS

## Planning Framework

### 1. Requirements Analysis
- What exactly needs to be built/changed
- User-facing behavior expected
- Which role(s) affected (customer/shop/admin)

### 2. Codebase Impact
- Which feature module(s) are involved
- Files to create vs modify
- Shared code affected (components, hooks, services, stores)
- API endpoints needed (existing or new)

### 3. Implementation Steps

Break into phases with clear checkboxes:

**Phase 1: Setup**
- [ ] Types and interfaces
- [ ] API service functions

**Phase 2: Core Logic**
- [ ] React Query hooks
- [ ] State management changes
- [ ] Business logic

**Phase 3: UI**
- [ ] Components
- [ ] Screens
- [ ] Navigation/routing

**Phase 4: Polish**
- [ ] Error handling
- [ ] Loading states
- [ ] Edge cases

### 4. Dependencies
- npm packages needed
- Backend API requirements
- Other features this depends on

### 5. Testing Strategy
- Manual testing steps
- Key scenarios to verify
- Edge cases to check

## Output
Provide a clear, actionable plan that can be followed step by step. Reference specific file paths in the codebase.
