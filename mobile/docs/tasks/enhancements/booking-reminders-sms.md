# Feature: Booking Reminders via SMS

**Status:** Open
**Priority:** Low
**Est. Effort:** Large (requires Twilio integration)
**Created:** 2026-03-10
**Updated:** 2026-03-10

---

## Overview

Send SMS reminders for upcoming appointments using Twilio.

## Note

**Defer to future sprint** - Email reminders already implemented.

## Requirements

1. Twilio account setup
2. SMS templates
3. Phone number validation
4. Customer opt-in for SMS notifications

## Implementation Steps

1. Install Twilio SDK
2. Create SMS service
3. Add phone number field to customer profile
4. Create SMS templates for reminders
5. Integrate with appointment reminder scheduler
